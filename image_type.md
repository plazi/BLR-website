## Displays

### Search result display

* image thumbnail
* image caption = title
* publication info = articleTitle + journalTitle + journalVolume + journalIssue + journalYear + articleDoi + articlePages + pages
* link to zenodo record = doi
* treatment flag, if belongs to a treatment
